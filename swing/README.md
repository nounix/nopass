# iPASS

iPASS is a password manager and generator which uses AES encryption and SHA-3 hashing.

# Build

## Docker:
- $ docker build -t ipass .
- $ ./ipass.sh

## Without docker:
- $ ./gradlew build gen
- $ ./ipass
